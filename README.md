# Saraswathan-portfolio
🚀 Inventory Management System (IMS)

A modern Inventory Management System built with ASP.NET Core Web API, Angular, SQL Server, and Azure. The application helps businesses manage products, warehouses, stock, purchase orders, sales, and user roles efficiently.

📌 Features

- User Authentication & Authorization (JWT)
- Role-Based Access Control (Admin, Manager, Employee)
- Product & Category Management
- Warehouse & Branch Management
- Purchase Orders
- Sales Orders
- Stock Transfer
- Inventory Tracking
- Dashboard & Reports
- Audit Logs
- Email Notifications
- File Uploads
- RESTful APIs
- Responsive Angular UI
